# LUIGISMANSIONCapture

## Controle

 Mode Flash :

 - joystick gauche : déplacement
 - joystick droit : direction flash
 - A Button : Flash
 - B button : if flash -> mode capture

 Mode Capture :

 - joystick gauche : direction du joueur
 - joystick droit : direction recul

 ## Fonctionnalité

 - Lennemi patrouille par défaut
 - FLasher l'ennemi le passe en mode stun (permet de le capturer) et il passe en fuite après 5 secondes
 - l'ennemi tourne autour du joueur en mode capture
 - le joueur doit diriger son joystick gauche vers l'ennemi et le droit à son opposé
 - la barre de capture augmente plus ou moins selon l'angle de direction du joystick gauche vers l'ennemi
